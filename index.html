electric-avenue by joshmcarthur

A Node.js frontend web service to ffmpeg, using HTML5, CSS3, and Redis

Electric Avenue is a simple application providing a flexible and customizable open-source web service for encoding a range of audio and video formats. It supports queueing files for encoding, running locally or on a remote server, and customizing presets to encode in different formats or with different configurations for different devices.

Dependencies

Node JS/NPM Redis ffmpeg with codecs

Install

git clone git@github.com:joshmcarthur/electric-avenue.git cd electric-avenue npm install node server.js
